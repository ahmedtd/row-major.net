Project 8 -- Triangular Meshes

Taahir Ahmed

Code is available as a git checkout (tag csce647-pr10): 
git://row-major.net/personal/ahmedtd/ballistae

Meshes are loaded from an obj file. Ray intersections are accelerated using a kd-tree built using a locally-greedy surface area heuristic. For the Stanford bunny (~5000 triangles), it takes approximately a minute to build the tree, and subsequent queries are only very slightly slower than queries against a sphere.
